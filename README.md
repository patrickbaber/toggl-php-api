toggl-php-api v0.1
==================

A PHP wrapper for the toggl time tracking API using cURL.

Notice
------

This is only a prototype. Don't use it in production environments.

Requirements
------------

- PHP 5.x
- PHP cURL extension

-----

See example.php for more details.
